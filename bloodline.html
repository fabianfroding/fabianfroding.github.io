Bloodline

A dialogue-based game prototype
×

A dialogue-based game created with the Deig editor. Features 4 different endings based on player choices, a custom-made map, and art and sprites from the Fire Emblem series.

Tools Used:
• DEIG
• Unity (For building the game)
• INKARNATE (For creating maps)
• KRITA (For modifying background and character images)
• SPRITERS RESOURCE (For ripping character images)
• DRAW.IO (For creating flowcharts for dialogues and location-transitions)
